# Animal Looks Explorer

A high-performance, minimalist, and visually immersive animal education web application designed specifically for toddlers (Ages 2-5).

## 🚀 Concept
The Animal Looks Explorer is built with a **"Toddler-First" UX philosophy**. It removes all traditional UI clutter and replaces it with a pure, sensory-driven interaction model. The goal is to encourage animal recognition and phonetic mimicry through high-contrast visuals and consistent narration.

## ✨ Key Features
- **Vibrant 3D Stylized Visuals**: Custom-generated, realistic 3D animals designed with high contrast to capture a toddler's attention.
- **Consistent Audio Narration**: High-quality narration in both **Bahasa Indonesia** and **English** (Slowed to 0.75x speed).
- **Sensory Feedback Loop**: 
    - **Interactive Bounce**: Animals pulse and react to touch.
    - **Spatial SFX**: Cinematic "Whoosh" sounds on swipe and "Magic Sparkle" sounds on tap.
- **PWA (Progressive Web App)**: Can be installed on any tablet or phone home screen to run in **Standalone Full-Screen Mode**.
- **Butter-Smooth Performance**: Optimized with hardware acceleration and asset preloading.

## 🛠 Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS 4
- **Deployment**: GitHub Pages via Automated GitHub Actions

## 📱 Optimization
- **Responsive Design**: Dynamically scales images to occupy 85-95% of the viewport.
- **Asset Preloading**: Preloads the next animal's image and voice for zero-latency navigation.
- **Static Export**: Built as a fully static site for instantaneous loading.
