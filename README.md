# Animal Looks Explorer

A minimalist, sensory-driven web application designed to help toddlers learn animal names through realistic visuals and bilingual audio.

## 👶 Features
- **Realistic Young Animals**: High-fidelity photography of baby animals (cubs, calves, foals) to teach true-to-life recognition.
- **Bilingual Narration**: Warm, clear voices in both **Bahasa Indonesia** and **English** (slowed to 0.75x speed).
- **Toddler-First Interaction**: Pure gesture-based navigation (swiping) with zero UI clutter.
- **Progressive Web App**: Optimized for full-screen use on tablets and phones.

## 🛠 Tech
- **Framework**: Next.js 16 (App Router)
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS 4

## 📱 Use
Simply open the web app on any device. Tap an animal to hear its name, or swipe left/right to explore.
